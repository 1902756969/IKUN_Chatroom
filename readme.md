# SSM实时聊天系统

## 一、技术栈

语言：Java 开发工具：IntelliJIDEA-2022.3

JDK版本：JDK8

项目管理工具：Maven3

前端技术：Html+ Css +JS+JQuery+bootstrap

后端技术：Spring+SpringMVC +netty4
